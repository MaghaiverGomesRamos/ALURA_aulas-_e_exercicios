# 📚 Alura

Este repositório reúne códigos feitos durante meus estudos em cursos e aulas.

O objetivo aqui é **praticar conceitos e acompanhar as implementações feitas durante as aulas**, registrando a evolução do aprendizado.

Alguns projetos podem aparecer em **diferentes versões ou etapas**, já que muitos cursos evoluem o mesmo projeto ao longo das aulas.

> Este repositório é apenas para **estudo e prática**. Projetos mais completos ou autorais ficam em repositórios separados.
